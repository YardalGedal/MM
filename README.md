## MMHash
MadMonday Hash

#### How to use?

```python
from MMHash import MMHash
hashsize = 256
print(MMHash('text', hashsize))
```

## MMCrypt
MadMonday Crypt

#### How to use?

```python
from MMCrypt import MMCrypt

crypted_text = MMCrypt().crypt('text')
encrypted_text = MMCrypt().encrypt(crypted_text)
```
